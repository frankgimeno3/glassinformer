# docs/

Optional specs. **Start with [docs/MAP.md](MAP.md)** for architecture, routing ("if you touch X → open Y"), and global decisions. For index and doc usage, see root [AGENTS.md](../AGENTS.md).

**In MAP.md** you will also find the **bar for local docs**: what a local doc must have (purpose, contract, flow, invariants, gotchas, how to change without breaking, tests) to deserve to exist. If a local doc cannot fill that, it should not be created; the information belongs in the hub or in the code.
